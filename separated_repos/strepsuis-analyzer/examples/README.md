# Examples Directory

This directory contains example datasets and outputs for StrepSuis Analyzer.

## Contents

### Input Data

- `Virulence.csv` - Example virulence factor data (located in `../data/`)

### Output Examples

The `outputs/` subdirectory contains example outputs generated by running the analyzer:

- `analysis_report.html` - Interactive HTML report
- `analysis_results.xlsx` - Excel workbook with multiple sheets
- `clustered_data.csv` - Data with cluster assignments
- `trait_associations.csv` - Statistical association results

## Running the Examples

### Basic Example

```bash
# From the strepsuis-analyzer directory
python StrepSuisPhyloCluster_2025_08_11.py --data data/Virulence.csv --output examples/outputs
```

### Docker Example

```bash
# Using Docker
docker-compose up
```

### Custom Parameters

```bash
# Specify number of clusters
python StrepSuisPhyloCluster_2025_08_11.py \
  --data data/Virulence.csv \
  --clusters 4 \
  --output examples/custom_outputs
```

## Understanding the Outputs

### HTML Report
- Open `outputs/analysis_report.html` in a web browser
- Interactive visualizations with Plotly
- Cluster distribution charts
- MCA biplots
- Top significant associations

### Excel Workbook
- Open `outputs/analysis_results.xlsx` in Excel or LibreOffice
- Multiple sheets:
  - **Clustered_Data**: Full dataset with cluster labels
  - **Cluster_Summary**: Statistics for each cluster
  - **Trait_Associations**: Statistical test results
  - **Metadata**: Analysis parameters

### CSV Files
- `clustered_data.csv`: Can be imported into other tools
- `trait_associations.csv`: Statistical results for further analysis

## Generating New Examples

To regenerate the example outputs:

```bash
# Clean old outputs
rm -rf examples/outputs/*

# Run analysis
python StrepSuisPhyloCluster_2025_08_11.py \
  --data data/Virulence.csv \
  --output examples/outputs \
  --seed 42

# Outputs will be created in examples/outputs/
```

## Notes

- Example outputs are generated with `--seed 42` for reproducibility
- Your results should match exactly if using the same seed
- File sizes may vary slightly across different systems
- HTML reports require a modern web browser to view
