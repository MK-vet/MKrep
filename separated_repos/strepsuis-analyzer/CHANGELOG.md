# Changelog

All notable changes to StrepSuisAnalyzer will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-15

### Added
- Interactive Streamlit web interface for comprehensive genomic analysis
- Statistical analysis module (correlations, hypothesis tests, meta-analysis)
- Phylogenetic analysis with tree visualization and metrics
- Machine learning clustering (K-Means, K-Modes, Hierarchical, DBSCAN)
- Advanced visualization with Plotly and matplotlib
- ETL operations (pivot tables, aggregations, window functions)
- Report generation to Excel and HTML formats
- Multi-format data support (CSV, Newick, Excel)
- Docker deployment with docker-compose
- Comprehensive test suite with 85%+ coverage
- Professional documentation suite

### Initial Features
- File upload interface for AMR, MIC, virulence, MLST, phylogenetic data
- Interactive widgets for parameter selection
- Real-time data visualization
- Export functionality for results and reports
- Responsive design for desktop and tablet
