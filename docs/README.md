# MKrep Documentation Portal

This directory contains the GitHub Pages documentation site for MKrep.

## Structure

- `index.html` - Main landing page
- `tools/` - Individual tool documentation pages
- `results/` - Example results and analysis outputs
- `demo-data/` - Demo datasets and download links
- `assets/` - Static assets (images, CSS, etc.)

## Accessing the Site

Once GitHub Pages is enabled, the site will be available at:
`https://mk-vet.github.io/MKrep/`

## Local Testing

To test the site locally:

```bash
cd docs
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.
