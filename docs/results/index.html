Understanding the Results

All MKrep tools generate three types of outputs:

HTML Reports

Interactive reports with:

  • DataTables with sorting, filtering, and export
  • Plotly visualizations (interactive charts)
  • Collapsible sections for easy navigation
  • Interpretation sections explaining results

Excel Workbooks

Structured data files with:

  • Multiple sheets (metadata, methodology, results)
  • All numerical results in tabular format
  • Chart index with PNG file references
  • Analysis parameters and timestamps

PNG Charts

Publication-quality figures:

  • High resolution (150+ DPI)
  • Professional styling
  • Ready for manuscripts
  • Organized in png_charts/ folder
