# Archived Documentation

This directory contains historical documentation files that have been archived as part of
documentation consolidation. These files document the development history and implementation
progress of the StrepSuis Suite.

## Archive Contents

### Implementation Summaries
- `IMPLEMENTATION_SUMMARY*.md` - Historical implementation progress notes
- `COMPLETION_SUMMARY.md` - Task completion records
- `IMPROVEMENTS_SUMMARY.md` - Improvement tracking
- `PR_SUMMARY.md` - Pull request summaries
- `WORKFLOW_FIX_SUMMARY.md` - Workflow fixes

### Module-Specific Documentation
- `CLUSTER_ANALYSIS_README.md` - Cluster analysis module docs (merged into main docs)
- `MDR_README.md`, `RUN_MDR_ANALYSIS.md` - MDR module docs
- `NETWORK_ANALYSIS_*.md` - Network analysis docs
- `PHYLOGENETIC_*.md` - Phylogenetic analysis docs

### Deployment & Separation Guides
- `MODULE_SEPARATION_*.md` - Module separation documentation
- `REPOSITORY_SEPARATION_GUIDE.md` - Repository restructuring guide
- `SEPARATED_MODULES_SUMMARY.md` - Separated modules overview
- `DEPLOYMENT_OPTIONS.md` - Historical deployment options
- `TOOL_PORTAL_GUIDE.md` - Tool portal implementation

### Feature & Output Documentation
- `EXCEL_REPORTS_README.md`, `EXCEL_REPORT_EXAMPLES.md` - Excel report details
- `APPLICATION_VARIANTS_GUIDE.md` - Application variant guide
- `COMPLETE_WORKFLOW_GUIDE.md` - Workflow documentation
- `PROJECT_SUMMARY.md` - Project overview
- `MODULE_VERIFICATION_SUMMARY.md` - Module verification status

### Other Historical Files
